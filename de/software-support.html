2. Vereinbarung

Als Softwareunternehmen gibt Worldpixel Software seinen Benutzern die Möglichkeit, Probleme, die mit der betreffenden Software auftreten, zu melden. Hierfür kann es notwendig sein, spezifische Daten an den Entwickler zu senden, die die Benutzeraktivität innerhalb der Software protokollieren und spezifische Systeminformationen sammeln. Diese Daten werden so weit wie möglich anonymisiert und anschließend per E-Mail an den Entwickler übermittelt.

Sollte es notwendig sein, andere personenbezogene Daten zu erheben, wird der Entwickler diese Daten sicher auf seinen Systemen speichern und sie nur so lange aufbewahren, wie es zur Behebung des jeweiligen Problems erforderlich ist. Personenbezogene Daten wie Namen und Kontaktdaten werden nur für die Lösung des technischen Problems verwendet und nicht für andere Zwecke wie beispielsweise Werbung.

Unter Umständen kann es auch notwendig, sein durch den Benutzer übermittelte Informationen an Dritte weiterzuleiten. Auch in diesem Fall werden diese so weit wie möglich anonymisiert, und nur notwendige Informationen mit Dritten geteilt. Sollten aufgrund unvorhergesehener Umstände personenbezogene Daten an Dritte weitergegeben werden müssen, wird der Benutzer vom Entwickler im Vorhinein um seine Zustimmung gebeten.
Sollten wir des Weiteren über die Veröffentlichung von Daten aus unserer Obhut (Daten-Leak) informiert werden, werden wir uns so schnell wie möglich an die betroffenen Benutzer wenden.

3. Was sind Ihre Rechte als Betroffene*r?

Daten die gesammelt, gespeichert, aktualisiert und gelöscht werden, werden in Übereinstimmung mit der DSGVO der EU verarbeitet. Wir möchten gerne sicher gehen, dass Sie sich über Ihre Rechte als Betroffene*r im Klaren sind:

  • Informationspflicht - Sie haben das Recht die bei uns (Worldpixel Software) über Sie gespeicherten Daten anzufordern.
  • Recht auf Berichtigung - Sie haben das Recht dazu, Daten die wir (Worldpixel Software) über Sie besitzen, korregieren oder vervollständigen zu lassen, sollten Sie glauben, dass es einer solchen Korrektur oder Vervollständigung bedarf.
  • Recht auf Vergessenwerden - Unter bestimmten Bedingungen haben Sie das Recht dazu, all Ihre Daten, die wir von Ihnen besitzen, bei uns (Worldpixel Software) löschen zu lassen. Daten die lokal auf Ihrem Gerät druch eine unserer Anwendungen gespeichert werden, lassen sich durch eine Deinstallation löschen.
  • Recht auf Einschränkung der Verarbeitung - Unter bestimmten Bedingungen haben Sie das Recht dazu, Worldpixel Software darum zu bitten die Verarbeitung Ihrer Daten einzuschränken.
  • Widerspruchsrecht - Unter bestimmten Bedingungen haben Sie das Recht dazu, Worldpixel Software die Verarbeitung Ihrer personenbezogenen Daten zu Untersagen.
  • Recht auf Datenübertragbarkeit - Unter bestimmten Bedingungen haben Sie das Recht dazu, von Worldpixel Software die Weiterleitung der sich in unserem Besitz befindlichen Daten über Sie an eine dritte Organisation oder direkt an Sie zu verlangen.

Hier bei Worldpixel Software ist uns Datenschutz wichtig und wir stehen gegen das unnötige Sammeln von Daten.
